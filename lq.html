LQ(linear quadratic) 문제

기간 [0,\, \tau] 동안 2차 형식으로 된 비용을 최소화하는 다음과 같은 형태의 문제를 LQ 문제, 또는 LQ 규제(LQ control)라 한다.

\min\int_{0}^{\tau }[\mathbf{x}^{\top}(t)\mathbf{Qx}(t) + \mathbf{u}^{\top}(t)\mathbf{Ru}(t)]dt + \mathbf{x}^{\top}( \tau )\mathbf{Hx}( \tau )

s.t \mathbf{x}'(t) = \mathbf{Ax}(t) + \mathbf{Bu}(t) \mathbf{x}

\mathbf{x}는 n×1 벡터, \mathbf{u}는 m×1 벡터, \mathbf{A}는 n×n 행렬, \mathbf{B}는 n×m 행렬, \mathbf{Q}, \mathbf{H}는 n×n positive semidefinite 대칭행렬, \mathbf{R}은 m×m positive definite 대칭행렬

\mathbf{x}^{\top}는 \mathbf{x}의 전치(transpose)를 의미

HJB 방정식은

-V{}_{t}= \min\mathbf{u}{\mathbf{x}^{\top}\mathbf{Qx} + \mathbf{u}^{\top}\mathbf{Ru} + V\mathbf{x}(\mathbf{Ax} + \mathbf{Bu})}, V(\mathbf{x},\tau) = \mathbf{x}^{\top}( \tau )\mathbf{Px}( \tau )

where V_{t} = \frac{\partial V(\mathbf{x},t)}{\partial t}, V_{\mathbf{x}} =\frac{\partial V(\mathbf{x},t)}{\partial \mathbf{x}}

HJB 방정식을 풀기 위해서는 가치함수를 추정해야 한다. 가치함수는 보통 목적식이나 말기함수의 형태를 따라 추정한다.

V(\mathbf{x},t) = \mathbf{x}^{\top}\mathbf{P}(t)\mathbf{x}로 추정하면, \mathbf{P}(t)는 대칭행렬

V{}_{t}(\mathbf{x},t) = \mathbf{x}^{\top}\mathbf{P}'(t)\mathbf{x}............[1]

V\mathbf{x}(\mathbf{x},t) = 2\mathbf{x}^{\top}\mathbf{P}(t) ..............[2]

이를 HJB 방정식에 대입하면

-\mathbf{x}^{\top}\mathbf{P}'(t)\mathbf{x} = \min\mathbf{u}{\mathbf{x}^{\top}\mathbf{Qx} + \mathbf{u}^{\top}\mathbf{Ru} + 2\mathbf{x}^{\top}\mathbf{P}(\mathbf{Ax} + \mathbf{Bu})} .................①

최소값을 구하기 위해 중괄호안을 \mathbf{u}에 대해 미분하면 1차조건은

2\mathbf{u}^{\top}\mathbf{R} + 2\mathbf{x}^{\top}\mathbf{\mathbf{P}\mathbf{B}} = 0

⇒ \mathbf{Ru} = -\mathbf{B}^{\top}\mathbf{Px}, (∵ \mathbf{R}, \mathbf{P}는 대칭행렬)

∴ \mathbf{u} = -\mathbf{R}{}^{-1}\mathbf{B}^{\top}\mathbf{Px}

⇒ \mathbf{u}^{\top} = -\mathbf{x}^{\top}\mathbf{\mathbf{P}\mathbf{B}\mathbf{R}}{}^{-1}

\mathbf{u}를 ①에 대입하면 \mathbf{u}는 최소화값이므로 min 연산자가 제거된다.

-\mathbf{x}^{\top}\mathbf{P}'\mathbf{x} = \mathbf{x}^{\top}\mathbf{Qx} + \mathbf{x}^{\top}\mathbf{\mathbf{P}\mathbf{B}\mathbf{R}}{}^{-1}\mathbf{\mathbf{R}\mathbf{R}}{}^{-1}\mathbf{B}^{\top}\mathbf{Px} + 2\mathbf{x}^{\top}\mathbf{P}(\mathbf{Ax} - \mathbf{\mathbf{B}\mathbf{R}}{}^{-1}\mathbf{B}^{\top}\mathbf{Px})

⇒ \mathbf{x}^{\top}\mathbf{P}'\mathbf{x} + \mathbf{x}^{\top}\mathbf{Qx} + \mathbf{x}^{\top}\mathbf{\mathbf{P}\mathbf{B}\mathbf{R}}{}^{-1}\mathbf{B}^{\top}\mathbf{Px} + 2\mathbf{x}^{\top}\mathbf{P}(\mathbf{Ax} - \mathbf{\mathbf{B}\mathbf{R}}{}^{-1}\mathbf{B}^{\top}\mathbf{Px}) = 0

⇒ \mathbf{x}^{\top}\mathbf{P}'\mathbf{x} + \mathbf{x}^{\top}\mathbf{Qx} - \mathbf{x}^{\top}\mathbf{\mathbf{P}\mathbf{B}\mathbf{R}}{}^{-1}\mathbf{B}^{\top}\mathbf{Px} + 2\mathbf{x}^{\top}\mathbf{PAx} = 0

⇒ \mathbf{x}^{\top}(\mathbf{P}'+ \mathbf{Q} - \mathbf{\mathbf{P}\mathbf{B}\mathbf{R}}{}^{-1}\mathbf{B}^{\top}\mathbf{P} + \mathbf{\mathbf{P}\mathbf{A}} + \mathbf{A}^{\top}\mathbf{P})\mathbf{x} = 0................[3]

이것이 항상 성립하기 위해서는

\mathbf{P}'+ \mathbf{Q} - \mathbf{\mathbf{P}\mathbf{B}\mathbf{R}}{}^{-1}\mathbf{B}^{\top}\mathbf{P} + \mathbf{\mathbf{P}\mathbf{A}} + \mathbf{A}^{\top}\mathbf{P} = 0

이는 \mathbf{P}'에 관한 Riccati 미분방정식이다. 이는 정형화된 해가 존재하므로 LQ 문제의 경우 항상 해가 존재한다.

예)

\min\int_{0}^{\top} {[y(t) ^{2} +u(t) ^{2} ]dt}

s.t y ' (t)=ay(t)+bu(t), a, b는 스칼라 상수

(풀이)

HJB 방정식은

- \frac{\partial V(y,\,t)}{\partial t} \,=\,\min_{u(t)\,} \left\{ y^{2} +u^{2} \,+\, \frac{\partial V(y,\,t)}{\partial y} (ay+bu) \right\}

가치함수를 V(y,\,t)\,=\,p(t)y^{2}으로 추정한다.

이를 HJB 방정식에 대입하면

-p ' y^{2} \,=\,\min_{u(t)\,} \left\{ y^{2} +u^{2} \,+\,2py(ay+bu) \right\}.................①

최소값을 구하기 위해 중괄호안을 u에 대해 미분하면, 1차 조건은

2u+2pyb\,=\,0 ⇒ u\,=\,-pyb

u를 ①에 대입하면

-p ' y^{2} \,=\,y^{2} +p^{2} y^{2} b^{2} \,+\,2py(ay-bpyb)

0\,=\,y^{2} (p ' +1+p^{2} b^{2} \,+\,2pa-2b^{2} p^{2} )

항상 성립하기 위해서는

p ' -b^{2} p^{2} \,+\,2ap\,+1\,=\,0 ..............②

이는 Riccati 미분방정식이다.

Riccati 미분방정식을 풀기 위해 이를 만족하는 한 특수해를 구한다.[4]

p(t)= \alpha, 상수라 하면 ②는

b^{2} \alpha^{2} \,-\,2a \alpha \,-1\,=\,0

∴ \alpha \,=\, \frac{a\pm \sqrt {a^{2} +b^{2}}}{b^{2}}

V(y,\,t)\, \geq \,0 ⇒ p(t)\, \geq \,0 ⇒ \alpha \, \geq \,0 이므로

∴ \alpha *\,=\, \frac{a+ \sqrt {a^{2} +b^{2}}}{b^{2}}

따라서 \alpha *는 Riccati 미분방정식을 만족하는 한 특수해가 된다.

z\,=\, \frac{1}{p- \alpha *} 라 하면, 공식에 따라[5]

\frac{dz}{dt} \,=\,-b^{2} \,-\,(2b^{2} \alpha *-2a)z

\,=\,-b^{2} \,-\,2z \sqrt {a^{2} +b^{2}}

⇒ \frac{1}{b^{2} \,+\,2z \sqrt{a^{2} +b^{2}} }\frac{dz}{dt} \,=\,-1

양변 적분하면

\frac{1}{2z \sqrt {a^{2} +b^{2}}} \,\ln (b^{2} \,+2z \sqrt {a^{2} +b^{2}} )\,=\,-t+C_{0}

⇒ b^{2} \,+2z \sqrt {a^{2} +b^{2}} \,=\,C_{1} \,\exp[-2z \sqrt {a^{2} +b^{2}} t]

⇒ z\,=\,C_{2} \,\exp[-2z \sqrt {a^{2} +b^{2}} t]- \frac{b^{2}}{2 \sqrt {a^{2} +b^{2}}}

∴ p(t)\,=\, \frac{1}{z} \,+\, \alpha *, where z\,=\,C_{2} \,\exp[-2z \sqrt {a^{2} +b^{2}} t]- \frac{b^{2}}{2 \sqrt {a^{2} +b^{2}}}, \alpha *\,=\, \frac{a+ \sqrt {a^{2} +b^{2}}}{b^{2}}

따라서 가치함수는 V(y,\,t)\,=\,p(t)y(t) ^{2}, 정책함수는 u(t)\,=\,-bp(t)y(t)
